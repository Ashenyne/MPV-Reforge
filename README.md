# MPV Player - Reforge


# Installation:

One of the nicest features of MPV is that it requires no installation and can be run from its own folder (portable).
1. Clone or download this repo.
2. Unzip to your desired location.
3. Unzip MPV Player.
4. And double clik MPV.exe
